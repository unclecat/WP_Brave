# Brave Love v0.4.5 发布说明

## ✨ 本次更新

### 🐛 Bug 修复
- **修复重复头像设置**：删除独立的"头像设置"部分，统一使用 Hero 区域设置
- **修复用户关联问题**：将下拉选择改为文本输入（输入用户ID），避免 Customizer 加载错误

---

## 📝 使用方法

### 设置头像
1. **外观 → 自定义 → Brave 主题设置 → Hero 区域**
2. **上传头像**：直接上传男生/女生头像图片
3. **关联 WordPress 用户**（可选）：
   - 在"男生关联 WordPress 用户ID"处填写用户ID
   - 在"女生关联 WordPress 用户ID"处填写用户ID
   - 用户ID查看方法：后台 → 用户 → 所有用户 → 悬停查看链接中的 ID

### 优先级
上传的头像 > WordPress 用户头像 > 默认头像
