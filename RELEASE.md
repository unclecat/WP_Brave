# Brave Love v0.6.28 发布说明

## 🔍 诊断版本

本版本用于排查浅色模式下页面Hero背景图不显示的问题。

### 新增诊断功能
- **Hero测试页面** (`page-test-hero.php`) 包含4个测试区域
- **测试3（诊断版）**: 橙色边框，背景图div带红色边框+绿色备用背景，遮罩层带蓝色边框
- **测试3b**: 粉色边框，测试波浪是否影响背景图

### 观察要点
| 颜色 | 含义 |
|-----|------|
| 橙色边框 | section区域 |
| 红色边框 | 背景图div存在 |
| 绿色背景 | 备用背景色（背景图未加载时显示） |
| 蓝色边框 | 遮罩层div存在 |
