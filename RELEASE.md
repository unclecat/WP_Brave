# Brave Love v0.4.4 发布说明

## ✨ 本次更新

### 👤 头像与 WordPress 用户关联
- **主题头像优先**：上传的头像优先显示
- **WordPress 头像 fallback**：未上传时使用 WordPress 用户头像
- **后台一键关联**：在 Hero 区域设置中选择 WordPress 用户
- **昵称同步**：自动获取 WordPress 用户昵称

---

## 📝 使用方法

### 设置头像（三种方式）

1. **方式一：上传头像（推荐）**
   - 外观 → 自定义 → Brave 主题设置 → Hero 区域
   - 上传男生/女生头像
   - 设置昵称

2. **方式二：关联 WordPress 用户**
   - 在 Hero 区域设置底部
   - 选择要关联的 WordPress 用户
   - 自动使用用户的头像和昵称

3. **方式三：混合使用**
   - 上传头像优先显示
   - 未上传时使用 WordPress 头像
   - 都没有则使用默认头像
