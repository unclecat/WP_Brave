# Brave Love v0.4.1 发布说明

## ✨ 本次更新

### 🎨 UI 优化
- **筛选器样式优化**
  - 修复双箭头问题，使用 CSS 绘制单箭头
  - 筛选栏改为圆角胶囊设计
  - 下拉菜单改为圆角卡片样式

### 🔧 逻辑优化
- **级联筛选逻辑改进**
  - 月份下拉框只在选择年份后显示
  - 日期下拉框只在选择月份后显示
  - 无数据时显示"暂无数据"提示

---

## 🚀 快速开始

1. 下载 `brave-love.zip`
2. WordPress 后台 → 外观 → 主题 → 上传主题
3. 激活主题
4. 访问 外观 → 自定义 → Brave 主题设置 进行配置
