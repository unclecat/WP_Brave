## 版本概述

本次发布是一个补丁版本，重点修复随笔说说页面在深色模式下用户名可读性过差的问题。

## 主要变更

### 1. 深色模式可读性修复
- 修复随笔说说列表卡片中用户名在深色模式下显示过暗的问题
- 修复随笔说说发布表单中当前用户名在深色模式下显示过暗的问题
- 同步提升时间文字在深色模式下的对比度，改善整体阅读观感

### 2. 发布与测试
- 保持 `tests/check-theme-simple.sh` 静态检查可通过
- 重新生成 `brave-love` 安装包

## 主要文件变更

| 文件 | 变更类型 | 说明 |
|------|----------|------|
| `assets/css/notes.css` | 修改 | 修复深色模式下用户名和时间文字对比度 |
| `style.css` | 修改 | 更新版本号 |
| `functions.php` | 修改 | 更新版本号 |

## 安装与更新

### 直接覆盖更新
1. 下载 `brave-love.zip`
2. 解压后通过 FTP 上传到 `wp-content/themes/brave-love/`
3. 覆盖所有文件
4. 清除浏览器缓存

### 注意事项
- 主题文件夹名称必须是 `brave-love`
- 主题设置和数据通常会保留
- 建议更新前备份当前主题

## 系统要求

- WordPress 6.0+
- PHP 7.4+
- 推荐 PHP 8.0+
