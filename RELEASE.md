## 📝 使用说明

### WordPress 用户ID是什么？
**不是用户名！** 是数字ID。

**查看方法**：
1. 后台 → 用户 → 所有用户
2. 鼠标悬停在用户名上
3. 看浏览器左下角链接：`user-edit.php?user_id=1`
4. `user_id=` 后面的数字就是ID

### 设置步骤
1. **设置主题头像**：外观 → 自定义 → Hero区域 → 上传头像
2. **关联用户**：在 Hero区域 底部填写 WordPress 用户ID
3. **随笔说说**：会自动显示对应头像
