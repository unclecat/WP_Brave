# Brave Love v0.2.0 发布说明

## 🎉 正式版发布

Brave Love 迎来 **v0.2.0 正式版**！这是一个专为情侣设计的 WordPress 主题。

---

## ✨ 新功能

### 💕 点点滴滴（时间轴）v2.0
- 全新卡片式设计，支持特色图片展示
- 独立摘要字段，列表页完整显示
- 按年份分组，横向滚动年份导航
- 锚点跳转，避免 URL 参数冲突

### 🌤️ 天气小组件
- App 风格卡片设计
- 支持 2-4 个城市天气
- 智能穿衣指南
- 点击展开详情弹窗

### ⏱️ 双计时器
- 恋爱正计时（已在一起多久）
- 纪念日倒计时
- 秒级实时跳动
- 自定义文案

## 💾 安装方法

1. 下载 `brave-love.zip`
2. WordPress 后台 → 外观 → 主题 → 添加新主题 → 上传主题
3. 激活主题
4. 访问 外观 → 自定义 → Brave 主题设置 进行配置
