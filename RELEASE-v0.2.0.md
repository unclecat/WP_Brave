# Brave Love v0.2.0 发布说明

## 🎉 正式版发布

经过多轮迭代优化，Brave Love 主题迎来 **v0.2.0 正式版**！这是一个专为情侣设计的 WordPress 主题，帮助你们记录爱情的点点滴滴。

---

## ✨ 核心功能

### 💕 点点滴滴（时间轴）
- 记录每一次见面、约会、旅行
- 自定义见面日期、地点、心情
- 支持添加摘要，列表页完整显示
- 按年份分组，快速导航筛选
- 精美卡片设计，支持特色图片

### 📜 恋爱清单
- 创建属于你们的 100 件小事
- 标记完成状态，记录完成日期
- 进度可视化展示

### 📷 甜蜜相册
- 使用 WordPress 原生编辑器上传照片
- PhotoSwipe 5 灯箱，支持移动端手势
- 可关联到点滴记录

### 📝 随笔说说
- 随时记录心情碎片
- 类似朋友圈的展示方式

### 🌤️ 天气小组件
- 支持 2-4 个城市天气
- App 风格卡片设计
- 智能穿衣指南
- 点击展开详情弹窗

### ⏱️ 双计时器
- 恋爱正计时（已在一起多久）
- 纪念日倒计时（距离下一个特别日子）
- 秒级实时跳动
- 自定义文案

---

## 🎨 设计特色

- **浪漫配色**：以粉色为主色调，营造甜蜜氛围
- **卡片式布局**：现代化的卡片设计，信息层次清晰
- **响应式设计**：完美适配手机、平板、电脑
- **流畅动画**：精心设计的过渡动效，提升用户体验
- **横向滚动**：年份、天气等支持横向滑动，操作便捷

## 🔧 技术栈

| 技术 | 说明 |
|------|------|
| PHP 7.4+ | 后端开发语言 |
| WordPress 6.0+ | 内容管理系统 |
| Bootstrap 5.3 | CSS 框架 |

## 🚀 安装方法

1. 下载 `brave-love.zip`
2. WordPress 后台 → 外观 → 主题 → 添加新主题 → 上传主题
3. 激活主题
4. 访问 外观 → 自定义 → Brave 主题设置 进行配置
