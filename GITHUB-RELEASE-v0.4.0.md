# Brave Love v0.4.0

## 📝 随笔说说全面升级

### ✨ 新功能

#### 🗓️ 级联日期筛选
- **年份筛选**: 从所有说说年份中选择特定年份
- **月份筛选**: 选择年份后，动态显示该年份有说说的月份  
- **日期筛选**: 选择月份后，动态显示该月有说说的日期
- **全部重置**: 一键清除所有筛选，查看全部说说

#### 🌊 瀑布流布局
- 响应式瀑布流设计，适配 1-3 列不同屏幕
- 纯 CSS 实现，无 JavaScript 依赖
- 卡片悬停动画效果，上浮和阴影增强

#### ✍️ 前台发布功能
- **仅限登录用户**: 安全的前台发布表单
- **心情表情选择**: 6 种心情可选 😊 😢 🥰 😌 🤔 😴
- **思念度星级**: 1-5 星滑动选择，直观表达思念程度
- **实时交互**: 点击心情和星星即时反馈

#### 🎴 说说卡片增强
- 作者头像和名称显示
- 发布日期时间
- 心情表情徽章
- 思念度星级展示

### 🚀 快速开始

1. 下载 `brave-wp-v0.4.0.zip`
2. 在 WordPress 后台 → 外观 → 主题 → 上传主题
3. 激活主题
4. 创建页面并选择"随笔说说"模板

### 📋 系统要求

- WordPress 6.0+
- PHP 7.4+
- Bootstrap 5.3 (CDN)
