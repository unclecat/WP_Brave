# Brave Love - WordPress 情侣主题

一个浪漫的 WordPress 情侣主题，灵感来源于 Typecho 主题 Brave。用于记录你们的点点滴滴、恋爱清单、甜蜜相册和随笔说说。

## 功能特性

### 🏠 首页 - 关于我们
- 恋爱正计时（实时显示已相恋天数、时、分、秒）
- 纪念日管理（支持倒计时和正数计时）
- 快捷入口卡片（5 大模块一键直达）
- 浪漫的 Hero 区域（背景图、双头像、跳动的爱心、波浪动画）

### 💖 点点滴滴
- 记录每次见面的时间、地点、做的事
- 时间轴展示，按年份筛选
- 支持心情标记和关联相册
- 移动端优化的时间轴布局

### 📜 恋爱清单
- 记录想一起做的 100 件事
- 完成状态标记和进度统计
- 支持分类筛选（旅行、美食、生活等）
- 完成后可上传纪念照片和记录

### 📷 甜蜜相册
- 按主题/事件管理照片
- 瀑布流展示，支持年份和标签筛选
- PhotoSwipe 灯箱，支持手势滑动和缩放
- 每张照片独立展示，可关联相关点滴

### 📝 随笔说说
- 朋友圈式的短内容发布
- 支持心情表情和多图上传
- 九宫格图片布局
- 点赞评论互动

### 💌 祝福留言
- 访客留言祝福板
- 优化的移动端评论表单
- QQ 邮箱自动识别头像

## 安装方法

1. 下载主题压缩包
2. 在 WordPress 后台 → 外观 → 主题 → 上传主题
3. 启用主题
4. 进入 外观 → 自定义 → Brave 主题设置，配置：
   - 情侣头像和昵称
   - 恋爱起始日期
   - 纪念日列表
   - 各页面入口

## 创建必要页面

主题需要创建以下页面并选择对应模板：

1. **首页** - 选择模板「首页 - 关于我们」
2. **点点滴滴** - 选择模板「点点滴滴」
3. **恋爱清单** - 选择模板「恋爱清单」
4. **甜蜜相册** - 选择模板「甜蜜相册」
5. **随笔说说** - 选择模板「随笔说说」
6. **祝福留言** - 选择模板「祝福留言」（建议关闭评论分页）

创建后在「Brave 主题设置 → 页面链接」中指定各页面。

## 使用自定义文章类型

### 添加点滴记录
1. 后台 → 点点滴滴 → 添加点滴
2. 填写标题、内容
3. 设置见面日期、地点、心情
4. 可选择关联已有相册
5. 设置特色图片作为封面

### 添加恋爱清单
1. 后台 → 恋爱清单 → 添加事项
2. 填写想做的事情
3. 完成后勾选「已完成」，设置完成日期
4. 可上传纪念照片和记录感想
5. 可使用「清单分类」进行归类

### 添加相册
1. 后台 → 甜蜜相册 → 添加相册
2. 填写相册主题和描述
3. 设置拍摄日期和地点
4. 在「照片管理」中上传多张照片
5. 可选择关联相关点滴

### 添加说说
1. 后台 → 随笔说说 → 添加说说
2. 选择心情表情
3. 填写短内容（无需标题）
4. 可选上传配图（最多9张）

## 移动优先设计

- 手机端单列布局，触摸友好的卡片尺寸
- 头像、波浪、计时器针对移动端优化
- 时间轴在手机上显示为单栏左线布局
- 相册瀑布流在手机端自动变为单列
- 所有按钮最小 44px 点击区域
- 图片懒加载，节省流量

## 技术栈

- Bootstrap 5.3 响应式框架
- PhotoSwipe 5 图片灯箱
- WordPress Customizer API 主题设置
- 自定义文章类型（CPT）存储数据
- 移动优先 CSS，支持暗色模式

## 自定义样式

在「Brave 主题设置 → 自定义代码 → 自定义 CSS」中添加：

```css
/* 示例：修改主色调 */
:root {
    --primary-color: #ff6b81;
}
```
