# Brave Love - WordPress 情侣主题

一个浪漫的 WordPress 情侣主题，记录你们的关于我们、点点滴滴、恋爱清单、甜蜜相册、随笔说说和祝福留言。

[![Version](https://img.shields.io/badge/version-0.7.9-blue.svg)](https://github.com/unclecat/WP_Brave/releases)
[![License](https://img.shields.io/badge/license-GPL%20v2-green.svg)](https://www.gnu.org/licenses/gpl-2.0.html)

## ✨ 主要功能

### 1. 💞 关于我们（About）
用一条更完整的故事线，记录一路走来的关键节点。

**功能特点：**
- 独立页面模板，适合梳理长期故事主线
- 支持自定义阶段、日期、摘要与正文内容
- 可关联到指定的「点点滴滴」文章

### 2. 💕 点点滴滴（Moments）
记录每一次见面，每一个瞬间。

**功能特点：**
- 时间轴展示，按年份分组
- 支持见面日期、地点、心情标记
- 自定义摘要，支持 HTML
- 分页功能（默认7篇/页，可自定义）
- 年份筛选导航

**使用说明：**
1. 后台 → 点点滴滴 → 添加新点滴
2. 填写见面日期、地点、选择心情
3. 上传特色图片或文章内容图片
4. 填写摘要（支持 HTML，如不填写自动截取内容）

### 3. 📜 恋爱清单（Love List）
记录你们一起想做的事、已完成的事。

**功能特点：**
- 卡片式布局，进度统计
- 完成状态标记，自动计算完成百分比
- 分类筛选
- 支持完成日期记录

**使用说明：**
1. 后台 → 恋爱清单 → 添加事项
2. 填写事项名称和描述
3. 完成后勾选"已完成"，填写完成日期
4. 可添加清单分类进行分组

**访问地址：** `/lists/`

### 4. 📷 甜蜜相册（Memories）
瀑布流照片墙，自动汇聚点点滴滴中的照片。

**功能特点：**
- CSS Columns 瀑布流布局
- 自动从点点滴滴文章提取照片
- PhotoSwipe 灯箱查看
- 显示照片信息（日期、地点、心情、摘要）
- 年份筛选
- 响应式适配（3/2/1列）

**使用说明：**
1. 在"点点滴滴"文章中上传照片（特色图片或内容图片）
2. 照片会自动显示在相册页面
3. 访问 `/memories/` 查看瀑布流相册

**后台设置：**
- 外观 → 自定义 → Brave 主题设置 → 相册设置
- 可设置每页照片数（默认12）
- 可设置是否显示照片信息

### 5. 📝 随笔说说（Notes）
记录心情碎片，日常随笔。

**功能特点：**
- 简洁的卡片式布局
- 支持心情表情
- 评论互动

**访问地址：** `/notes/`

### 6. 💌 祝福留言（Blessing）
接收来自亲朋好友的祝福。

**功能特点：**
- 自定义评论表单
- 内置卡通头像池
- 友好的时间显示

### 7. ⏱️ 双计时器
首页显示恋爱正计时 + 纪念日倒计时。

**功能特点：**
- 恋爱正计时：精确到秒，显示天数、小时、分钟、秒
- 纪念日倒计时：显示距离下一个纪念日的剩余时间
- 自定义文字和起始时间

**设置方法：**
后台 → 外观 → 自定义 → Brave 主题设置 → 基本信息
- 设置恋爱起始日期时间（精确到分钟）
- 设置下一个纪念日日期时间和名称
- 自定义计时器文字

### 8. 🌤️ 天气小组件
首页显示多城市天气信息。

**功能特点：**
- App 风格卡片设计
- 支持多个城市，自定义增减
- 点击展开详情弹窗
- 显示逐小时趋势、日照、紫外线和穿搭建议
- 使用 Open-Meteo 免费 API

**设置方法：**
后台 → 设置 → 天气管理
- 添加城市名称和坐标
- 设置更新频率

### 9. ✨ 星星旋转动画
首页 Hero 区域头像周围的星星环绕动画。

**效果：**
- 6 颗星星围绕头像旋转
- 白色星星带淡黄色光晕
- 8 秒一圈匀速旋转
- 星星独立闪烁动画
- 支持手机、平板、PC 响应式

## 🚀 安装方法

### 方法1：直接上传
1. 下载 `brave-love.zip`
2. 后台 → 外观 → 主题 → 添加新主题 → 上传主题
3. 启用主题

### 方法2：FTP 上传
1. 解压 `brave-love.zip`
2. 将文件夹上传到 `/wp-content/themes/`
3. 后台 → 外观 → 主题 → 启用

---

## ⚙️ 初始配置

### 第一步：刷新固定链接
安装后必须执行：
后台 → 设置 → 固定链接 → 点击「保存更改」（不修改内容）

### 第二步：创建必要页面
创建以下页面并选择对应模板：

| 页面名称 | 模板 | 访问地址 |
|---------|------|---------|
| 首页 | 首页 - 关于我们 | `/` |
| 关于我们 | 关于我们 | `/about-us/` |
| 点点滴滴 | 点点滴滴 | `/moments/` |
| 恋爱清单 | 归档页，无需单独页面 | `/lists/` |
| 甜蜜相册 | 甜蜜相册 | `/memories/` |
| 随笔说说 | 随笔说说 | `/notes/` |
| 祝福留言 | 祝福留言 | `/blessing/` |

**注意：** 恋爱清单不需要创建普通页面，直接使用归档页 `/lists/`

### 第三步：主题设置
后台 → 外观 → 自定义 → Brave 主题设置

**基本信息：**
- 恋爱起始日期时间
- 恋爱计时器文字
- 下一个纪念日信息

**页面链接：**
- 选择各个页面对应的页面

**入口图标：**
- 自定义各个入口的图标（支持 Emoji）

## 🎯 常见问题

### Q: 安装后页面 404？
A: 后台 → 设置 → 固定链接 → 点击「保存更改」刷新重写规则。

### Q: 相册页面没有照片？
A: 需要在「点点滴滴」文章中添加照片（特色图片或内容图片）。

### Q: 如何清理旧相册数据？
A: 后台 → 点点滴滴 → 相册数据管理，可以批量删除或转换为点滴。

### Q: 如何修改页面网址？
A: 编辑页面 → 右侧「永久链接」→ 修改 slug。

### Q: 星星动画不显示？
A: 确保使用最新版本，清除浏览器缓存。
