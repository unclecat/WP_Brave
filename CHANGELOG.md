# 更新日志 (Changelog)

所有版本的更新记录。

---

## [v0.1.17] - 2026-03-30

### 修复
- **年份筛选连接终止**: 彻底解决年份筛选导致的网站连接问题
  - 将参数名从 `year` 改为 `filter_year`，避免与 WordPress 日期系统冲突
  - 完全移除可能导致问题的重定向代码

## [v0.1.16] - 2026-03-30

### 修复
- **年份筛选重定向循环**: 修复点击年份筛选时"重定向次数过多"的错误
  - 移除导致循环的重定向逻辑
  - 仅禁用 canonical 重定向避免 URL 被修正

## [v0.1.15] - 2026-03-30

### 修复
- **年份筛选 404 问题**: 修复点点滴滴页面按年份筛选时显示 404 的问题
  - 添加 `brave_fix_moment_year_redirect` 函数防止 WordPress 错误解析 URL
  - 禁用 canonical 重定向避免冲突

### 优化
- **时间轴样式全面重设计**:
  - 优化时间轴线：渐变色彩 + 更粗的线条
  - 时间点：渐变背景 + 发光边框效果
  - 日期标签：添加 📅 图标 + 渐变背景
  - 内容卡片：更大的圆角 + 悬停动效
  - 新增 `.timeline-header` 布局（标题 + 心情分开）
  - 地点标签：独立区域 + 虚线分隔 + 📍 图标
  - 图片网格：改为 4 列 + 悬停放大效果
  - 空状态：独立样式 + 图标淡化效果

---

## [v0.1.14] - 2026-03-30

### 优化
- **天气卡片样式**: 改为 App 风格小方块设计
  - 固定尺寸 110×140px，大圆角 16px
  - 根据天气自动切换渐变背景色（晴/云/雨/雪）
  - 上下布局：城市名 + 天气图标 + 温度 + 描述
  - 卡片阴影随天气变化，更有层次感

---

## [v0.1.13] - 2026-03-30

### 重构
- **天气组件全新设计**:
  - 改为卡片式横向滚动布局，与"特别的日子"风格一致
  - 支持添加 2-4 个关心地区的天气
  - 卡片仅显示城市和温度，点击展开详情弹窗
  - 详情页显示：温度、天气状况、体感、湿度、风速、穿衣指南
  - 新增独立后台管理页面：设置 → 天气城市

---

## [v0.1.12] - 2026-03-30

### 新增
- **天气小组件**: 基于 Open-Meteo API 的免费天气显示
  - 显示当前温度、天气状况、体感温度、湿度、风速
  - 智能穿衣指南：根据温度和天气自动推荐穿衣建议
  - 支持自定义城市（可在 Customizer 中设置经纬度）
  - 每 30 分钟自动更新

## [v0.1.0] - 2026-03-29

### 初始版本
- **6 个页面模板**: 首页、时间轴、恋爱清单、相册、说说、祝福板
- **4 个自定义文章类型**: 时刻 (moment)、清单项 (list_item)、相册 (album)、说说 (note)
- **响应式设计**: 基于 Bootstrap 5.3，移动端优先
- **灯箱功能**: PhotoSwipe 5 集成
- **主题自定义**: 支持通过 Customizer 设置主题色、封面图等
