## ✨ 新功能

### 核心模块
- 💕 **首页/关于我们** - 恋爱计时器、纪念日管理、快捷入口
- 💖 **点点滴滴** - 时间轴记录每次见面
- 📜 **恋爱清单** - 100件事进度追踪
- 📷 **甜蜜相册** - 瀑布流照片管理
- 📝 **随笔说说** - 朋友圈式短内容
- 💌 **祝福留言** - 访客留言板

### 设计特性
- 📱 移动优先响应式设计
- 🌊 SVG 波浪动画 + 跳动爱心
- 🖼️ PhotoSwipe 5 图片灯箱
- 📅 纪念日倒计时/正计时
- 📊 恋爱清单环形进度图
- 🎨 CSS 变量支持自定义主题色

### 技术特性
- 4 个自定义文章类型 (CPT)
- 完整的后台管理界面
- 纪念日独立管理页面
- WordPress Customizer 主题设置
- 图片懒加载优化
- 多图上传和拖拽排序

## 🚀 快速开始

### 安装

1. 下载 `brave-love-v0.1.zip`
2. WordPress 后台 → 外观 → 主题 → 上传主题
3. 启用主题
4. 按照 README.md 完成配置

### 配置步骤

```
1. 设置 → 固定链接 → 保存
2. 外观 → Brave 主题设置 → 填写基本信息
3. 后台 → 设置 → 纪念日 → 添加纪念日
4. 创建页面并选择模板
5. 主题设置 → 页面链接 → 指定各页面
```
